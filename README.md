# Final Project

## Emotion Detection Web Application

This Final Project is an AI-based emotion detection web application built using IBM Watson NLP and Flask.

## Features

* Detects emotions from user input text
* Identifies dominant emotion
* Includes unit testing for validation
* Implements error handling for invalid inputs
* Uses static code analysis for code quality

## Technologies Used

* Python
* Flask
* IBM Watson NLP
* unittest
* pylint

## Description

The application takes user input as text and analyzes it using Watson NLP to detect emotions such as joy, anger, sadness, fear, and disgust. The results are displayed through a Flask-based web interface.
